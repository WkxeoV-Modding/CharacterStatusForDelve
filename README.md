# CharacterStatusForDelve
A simple addon to your avatar. Meant specifically for the Delve Server.

It is a simple script that you simply drag and drop into your Avatar folder.
Accessed via the Action Wheel! If you did not already have an Action Wheel page set up, this one will automatically make one.

Immediately when you load into the game you will see new text above your nameplate. It will be 'OOC | ???'' that is how you know it is working.

In the action wheel you will have a Character menu. Inside will be 2 buttons (if you had an existing action page it will be 3 buttons)
The status button changes the OOC part of the new text. By Left or Right Clicking you can change between the three statuses. OOC/IC/AFk in that order.

The Name Revealed? button changes the ??? into your name and vice versa. However, when clicked you will see it changes to Steve. This is on purpose.
Type in chat '$nameplate set default' then add your name after! It supports multiple words!

Specific to Delve, there is a feature in which your nickname on the server (the tab list) will auto update. However, a setting needs to be turned on.
I do not recommend.
